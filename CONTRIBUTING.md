# Contribution Guidelines

## Commit authorship and contribution standards

Contributions are evaluated on the quality of the code and nothing else. Whether a change was written entirely by hand, produced with LLM assistance, or generated by an autonomous agent is irrelevant to how it is reviewed. Humans and LLMs alike produce both good and bad code; the commit is what matters.

Every commit in this repository must be authored by a human being. The `author` and `committer` fields of every git commit object must identify a real, individual person. No LLM, autonomous agent, or automated system — regardless of the interface, wrapper, or tooling used to invoke it — may appear in either field, without exception.

LLM and agent assistance during development is permitted. When such tools contribute to the work in a commit, the human programmer who operated them is the sole author of record and bears full responsibility for all changes in that commit.

The `Co-authored-by` trailer is reserved for **human contributors only**.

### Disclosing LLM or agent assistance

To disclose which LLM or agent assisted in producing a commit, add one `Assisted-by` trailer per tool to the commit message, using the model name and version as the value.

Examples:
```plain
Assisted-by: claude-sonnet-4-6
Assisted-by: kimi-k2p5
```
